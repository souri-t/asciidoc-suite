本マニュアルは以下の方を対象としています：

* サンプルシステムを初めてご利用になる方
* 基本的なコンピューター操作ができる方
* Webブラウザの基本的な使い方をご存知の方

[NOTE]
====
本マニュアルでは、Windows 10/11とGoogle Chrome/Microsoft Edgeでの操作を基準に説明しています。
====
