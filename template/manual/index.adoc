[appendix]
== よくある質問（FAQ）

=== Q1: ログインできません
**A:** パスワードを忘れた場合は、ログイン画面の「パスワードを忘れた方」をクリックしてください。

=== Q2: データが保存されません
**A:** ブラウザの設定でCookieが有効になっているかご確認ください。

=== Q3: 画面が正しく表示されません
**A:** ブラウザのキャッシュをクリアして、ページを再読み込みしてください。
